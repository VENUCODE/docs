## Knowledge Base Configuration

**Manage data resources and knowledge base settings**

Configure your knowledge base by managing data resources, viewing content, and customizing settings for optimal performance.

### Data Resources Management

**Adding Data Resources**

- **Website URLs**: Add new websites to scrape and process
- **Document Uploads**: Upload additional PDF, DOC, or other supported files
- **CSV Data**: Import new tabular data or Q&A pairs
- **Manual Content**: Add custom text content and Q&A pairs

**Viewing Data Resources**

- **Content Preview**: View extracted text and processed content
- **Source Information**: See original URLs, file names, and upload dates
- **Processing Status**: Check training and indexing progress
- **Content Quality**: Review extracted content for accuracy

**Managing Data Resources**

- **Edit Sources**: Modify existing data source configurations
- **Delete Sources**: Remove unwanted data resources
- **Re-process**: Retrain specific data sources
- **Bulk Operations**: Manage multiple resources simultaneously

### Configuration Options

**Basic Settings**

- **Knowledge Base Name**: Update the display name
- **Description**: Modify the knowledge base description
- **Tags**: Add or edit categorization tags
- **Access Permissions**: Set visibility and sharing options

**Advanced Settings**

- **Search Configuration**: Customize search behavior and ranking
- **Content Filtering**: Set up content inclusion/exclusion rules
- **Update Frequency**: Configure automatic content refresh settings
- **Performance Tuning**: Optimize for specific use cases

### Training and Processing

**Training Status**

- **Current Status**: View real-time training progress
- **Training History**: See previous training sessions and results
- **Error Logs**: Review any processing errors or warnings
- **Performance Metrics**: Monitor training speed and quality

**Manual Actions**

- **Retrain**: Trigger manual retraining of all content
- **Partial Retrain**: Retrain specific data sources only
- **Test Queries**: Validate knowledge base responses
- **Export Data**: Download processed content and settings

<Tip>
  **Regular Updates**: Keep your knowledge base current by regularly updating
  data sources and retraining when content changes significantly.
</Tip>

<Warning>
  **Data Quality**: Ensure your source content is accurate and up-to-date before
  training, as this directly impacts the quality of AI Agent responses.
</Warning>
