<Steps>
<Step title="Click AI Agent Templates" icon="grid">
  In the support dropdown menu, click on `AI Agent Templates` to access our template library.
  
  <Check>
  This opens the TARS AI Agent marketplace in a new browser tab.
  </Check>
</Step>

<Step title="Explore AI Agent template marketplace" icon="store">
  The AI Agent marketplace opens with over 1000+ pre-built templates:
  <Frame>
    <img
      src="/images/dashboard/support/templates-page.png"
      alt="AI Agent Templates page"
    />
  </Frame>
  **Marketplace overview:**

  - **1000+ templates**: Extensive library of ready-to-use AI Agent templates
  - **Search functionality**: Use the search bar to find specific Agent types
  - **Category filters**: Browse by industry, function, or use case at the top
  - **Template cards**: Each template shows description, features, and action buttons

  **Template cards include:**

  - Template name and description
  - Key features and capabilities
  - `Use this template` button for direct creation
  - `Try AI Agent` button for testing the template
  <Info>
    **Marketplace Description**: "AI Agent Marketplace Tailored to Your Needs" -
    designed to help you find the perfect AI solution for your business.
  </Info>
</Step>

<Step title="Select and use template" icon="play">
  Choose a template that fits your needs:
  <Frame>
<img src="/images/dashboard/support/use-template.png" alt="use the template to start building your agent" />
</Frame>    
  **Template options:**
  - **Try AI Agent**: Click `Try AI Agent` to interact with the template and see how it works
  
  **Template detail page:**
  - **Interactive demo**: Test the Agent's capabilities and responses
  - **Description**: Detailed information about the template's purpose and features
  - **Video demonstration**: Watch how to build and customize the template
  - **Creation options**: Choose to use the template or create from scratch
  - **Use template**: Click `Use this template` to create your own Agent based on the template
  
  <Tip>
  **Template testing**: Use "Try AI Agent" first to understand how the template works before creating your own version.
  </Tip>
</Step>
</Steps>
