**API Key Authentication**

The most common and straightforward authentication method for connecting tools to your AI agents.

<Steps>
<Step title="Generate API key">
  - Log into your tool's admin panel or settings
  - Navigate to API settings or developer section
  - Click **"Generate API Key"** or **"Create New Key"**
  - Copy the generated API key (keep it secure)
</Step>

<Step title="Configure in Tars">
  - Go to Tools section in your Tars dashboard
  - Find the tool you want to connect
  - Click **"Connect"** or **"Add Integration"**
  - Select **"API Key"** as authentication method
</Step>

<Step title="Enter credentials">
  - Paste your API key in the **"API Key"** field
  - Add any additional required fields (if applicable)
  - Click **"Test Connection"** to verify
</Step>

<Step title="Complete setup">
  - Review the connection details
  - Click **"Save"** to complete the integration
  - Your tool is now connected and ready to use
</Step>
</Steps>

**Common tools using API Key:**
- **HubSpot**: Marketing and CRM platform
- **Stripe**: Payment processing
- **SendGrid**: Email delivery service
- **Zapier**: Workflow automation
- **Airtable**: Database and project management

<Note>
**Security**: API keys are encrypted and stored securely. Never share your API keys publicly or commit them to version control.
</Note>

<Warning>
**Best Practice**: Regularly rotate your API keys for enhanced security. Most tools allow you to generate new keys and revoke old ones.
</Warning>
