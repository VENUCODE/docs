Creating an agent from a template gives you a head start with pre-built workflows, proven conversation patterns, and industry-specific configurations that you can customize for your specific use case.

### What you get

**Proven foundation**

- **Industry expertise**: Templates built with best practices for specific industries
- **Pre-built workflows**: Ready-to-use conversation flows and decision trees
- **Optimized responses**: Tested and refined response patterns
- **Quick deployment**: Get your agent up and running in minutes
- **Customization flexibility**: Modify any aspect to match your specific needs

### Perfect for

- **Quick start**: When you need to deploy an agent quickly
- **Industry standards**: When you want to follow proven patterns for your industry
- **Learning**: When you want to understand how successful agents are structured
- **Consistency**: When you want to maintain consistent quality across multiple agents
- **Time constraints**: When you have limited time for agent development

### Step-by-step creation process

<Steps>
<Step title="Select template and preview">
  In the Create Agent section, browse through the available templates and click `See Preview` on the template card you want to explore.
  
  <Frame>
  <img src="/images/dashboard/agent/template/step-1-click-see-preview.png" alt="Template selection page showing available templates with See Preview buttons" />
  </Frame>
</Step>

<Step title="Review template details and video demonstration">
  View the template details including name, description, and tools used in the
  template.
  <Frame>
    <img
      src="/images/dashboard/agent/template/step-2-template-preview.png"
      alt="Template information section showing name, description, and tools used"
    />
  </Frame>
  Utilize the video demonstration to gain insights into the template's capabilities and conversation flow.

  **Video Content:**
  - **Building Process**: Provides a step-by-step guide on constructing the template.
  - **Functionality Demonstration**: Illustrates the template in action.
  - **Best Practices**: Explains the rationale behind template design decisions.
</Step>

<Step title="Test interactive agent">
  Interact with the live chatbot widget to test the template's capabilities and
  conversation flow.
  <Frame>
    <img
      src="/images/dashboard/agent/template/step-2-template-interactive.png"
      alt="Interactive template preview with live chatbot widget for testing"
    />
  </Frame>
  **Interactive testing:**
  - **Live interaction**: Chat with the template to see how it responds.
  - **Functionality testing**: Test different conversation scenarios and edge cases.
  - **Response evaluation**: Assess the quality and relevance of template responses.
</Step>

<Step title="Review step-by-step guide">
  Read the comprehensive step-by-step setup guide for implementing the template.
  <Frame>
    <img
      src="/images/dashboard/agent/template/step-2-step-by-step-setup-guide.png"
      alt="Step-by-step setup guide for template implementation"
    />
  </Frame>
  **Setup guide includes:**
  - **Implementation steps**: Detailed instructions for setting up the template
  - **Configuration details**: How to customize the template for your needs
  - **Best practices**: Recommendations for optimal template usage
  - **Troubleshooting**: Common issues and solutions
</Step>

<Step title="Use template and create agent">
  After reviewing the template details and testing the preview, click `Use Template` to create your agent.
  
  <Frame>
  <img src="/images/dashboard/agent/template/step-4-template-in-canvas.png" alt="Template-based agent created in canvas ready for customization" />
  </Frame>
  
  **Agent creation:**
  - **Template applied**: Your agent is created with all template configurations
  - **Ready for customization**: Agent is ready in the canvas for further modifications
  - **Immediate functionality**: Agent works immediately with the template's pre-built workflows
</Step>
</Steps>

<Tip>
